# 聲明與出處標示

本 repository 包含來源 metadata、整理後的風格 profile，以及從 Fandom「香港網絡大典」抓取的原始 wikitext。用途是學習和研究香港網絡潮文的結構、節奏和改寫方式。

本專案只供學習和研究用途。語料和 profile 旨在協助理解文字風格、文章結構及轉化模式，不應用作取代原始來源出處，也不應將複製內容誤導成自己的原創作品。

主要來源索引：

https://evchk.fandom.com/zh/wiki/%E6%BD%AE%E6%96%87

## 上游出處

`research/raw-corpus/` 下的 raw corpus 檔案歸功於 Fandom「香港網絡大典」的貢獻者。每個 raw file 都包含：

- 文章標題
- 來源 URL
- 抓取時間
- 來源修訂時間，如可取得

`research/raw-corpus/manifest.json` 亦記錄每篇文章的本地檔案路徑和來源 URL。
